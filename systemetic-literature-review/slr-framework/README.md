## Quick Start Guide

### For New SLR Projects

1. **Initial Setup**
   ```bash
   # Copy framework templates to your project folder
   /dashboard-init "Your Project Title" "Research Topic" "Team Lead Name"
   ```

2. **Framework Orientation**
   - Read `sllr_plan.md` for methodology overview
   - Review `slr_task.md` for detailed task breakdown
   - Set up `automation-dashboard.md` for tracking

3. **Begin Planning**
   - Start with Phase 0 (Pre-Planning) tasks
   - Use automated tracking to monitor progress
   - Leverage AI recommendations for optimal sequencing

### Daily Operations

1. **Progress Updates**
   ```bash
   /update-progress    # Mark completed tasks
   /health-check       # Identify blockers and risks
   /recommendations    # Get AI-powered next steps
   ```

2. **Reporting**
   ```bash
   /generate-report daily|weekly|milestone
   /export-excel       # Create spreadsheet reports
   /status            # Show current project status
   ```

## Best Practices

### Framework Usage
1. **Follow Sequentially**: Use phases in order for best results
2. **Document Everything**: Maintain detailed notes and decisions
3. **Update Regularly**: Keep task status current for accurate tracking
4. **Review Recommendations**: Validate AI suggestions before implementation

### Team Collaboration
1. **Shared Dashboard**: Ensure all team members have dashboard access
2. **Regular Updates**: Daily status updates maintain accuracy
3. **Clear Roles**: Assign specific responsibilities to team members
4. **Communication Log**: Document team decisions and discussions

### Quality Assurance
1. **Compliance Checks**: Regular verification against Seven Pillars
2. **Peer Review**: Have team members review each other's work
3. **Documentation**: Maintain complete audit trail of decisions
4. **Validation**: Cross-check automated recommendations
