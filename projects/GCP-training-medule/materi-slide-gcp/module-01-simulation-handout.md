- **PI harus bersertifikat CUKB** (valid 5 tahun)
- **Persetujuan ET WAJIB** sebelum PPUK
- **Perubahan protokol** = Amendment + ET approval
- **Electronic systems** harus validated

## 🎯 **Decision Tree Quick Guide**

### **IF** patients already present:
```
→ Prioritize patient care & communication
→ Screen properly (even if manual)
→ Document everything
→ Explain delay professionally
```

### **IF** protocol violation clear:
```
→ Stop immediately
→ Report to ET
→ Document reasons
→ Plan corrective actions
```

### **IF** technology not ready:
```
→ Use manual alternatives
→ Don't compromise on quality
→ Communicate limitations
```

## 💡 **Critical Thinking Prompts**
