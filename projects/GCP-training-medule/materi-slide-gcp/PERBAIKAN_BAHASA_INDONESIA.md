# Pedoman Perbaikan Bahasa Indonesia
## Untuk Semua Modul Slide GCP Training

### **Perubahan Utama yang Diterapkan:**

#### **1. Ganti Istilah Asing dengan Padanan Indonesia:**

| ❌ TIDAK BOLEH | ✅ WAJIB DIGUNAKAN |
|---------------|-----------------|
| GCP | CUKB |
| Investigator | Peneliti |
| Sponsor | Sponsor |
| IRB/IEC | ET/KE |
| Subject | Subjek/Partisipan |
| Clinical trial | Uji klinis |
| Safety | Keamanan |
| Documentation | Dokumentasi |
| Protocol | Protokol |
| Research | Penelitian |
| Team | Tim |
| Quality | Kualitas |
| Training | Pelatihan |
| Management | Manajemen |
| Process | Proses |
| Requirements | Persyaratan |
| Compliance | Kepatuhan |
| Assessment | Penilaian |
| Implementation | Implementasi/Pelaksanaan |
| Timeline | Garis waktu |
| Duration | Durasi |
| Activity | Aktivitas |
| Resources | Sumber daya |
| Experience | Pengalaman |
| Qualifications | Kualifikasi |
| Responsibilities | Tanggung jawab |

#### **2. Struktur Kalimat Indonesia:**

**Contoh perubahan:**
- ❌: "Investigator harus memiliki proper training"
- ✅: "Peneliti harus memiliki pelatihan yang tepat"

- ❌: "The protocol should be implemented correctly"
- ✅: "Protokol harus dilaksanakan dengan benar"

- ❌: "Documentation requirements must be met"
- ✅: "Persyaratan dokumentasi harus dipenuhi"

#### **3. Ejaan dan Tanda Baca EYD Edisi Kelima:**

- **"Dapat digunakan"** (bukan "dapat dipakai")
- **"Maka"** untuk menghubungkan klausa yang sejajar
- **"Pertanyaan:"** dengan titik dua untuk daftar
- **"Contoh:"** dengan titik dua untuk ilustrasi
- **"Penting:"** dengan titik dua untuk penekanan

#### **4. Format Speaker Notes:**

**Ubah dari:** `<!-- SPEAKER NOTES: -->`
**Menjadi:** `<!-- CATATAN PEMBICARA: -->`

#### **5. Referensi:**

**Ubah dari:** `**Referensi:**`
**Menjadi:** `**Rujukan:**`
