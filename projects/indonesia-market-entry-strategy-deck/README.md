# Indonesia CRO Market Entry Strategy Deck

An interactive slide presentation designed for executive decision-making regarding CRO (Contract Research Organization) market entry in Indonesia. This application features a phase-gate assessment framework covering market, regulatory, operational, and financial dimensions.

## Features

- **Interactive Slide Deck**: Keyboard navigable presentation (Arrow keys/Space).
- **Dynamic Charts**: Financial projections and risk matrix visualizations using Recharts.
- **Decision Calculator**: Interactive weighted scoring matrix for Go/No-Go decisions.
- **Executive UI**: High-impact design tailored for CEO-level presentations.

## Tech Stack

- **Framework**: React 19 (TypeScript)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Visualization**: Recharts

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm start
   ```
