# Indonesia News Briefing Skill

Provides professional daily news briefings about Indonesia's top developments from the country's most credible national news sources. Perfect for business executives, policymakers, and professionals who need to stay informed about current events affecting Indonesia's business and policy landscape.

## Files Structure

```
indonesia-news-briefing/
├── SKILL.md                    # Main skill definition and workflow
└── examples/                      # Practical examples and templates
    ├── briefing-example-2024-12-02.md     # Complete daily briefing example
    ├── executive-summary-example.md         # Executive summary format example
    ├── business-impact-example.md            # Business impact analysis example
    └── policy-analysis-example.md             # Policy analysis example
```

## Usage

1. **Upload Skill**: Install `indonesia-news-briefing.zip` via Claude Desktop Settings → Capabilities → Skills → Upload skill

2. **Request Briefing**: Ask "Berikan saya briefing berita terkini Indonesia untuk hari ini"

3. **Load Examples**: When helpful, request specific examples:
   - "Tunjukkan contoh briefing lengkap"
   - "Load briefing example 2 Desember 2024"
   - "Tunjukkan contoh analisis dampak bisnis"

## Skill Capabilities

✅ **Multi-Source News Gathering**: Brave Search (general overview) + Tavily Search (specific details)
✅ **Professional Analysis**: Business impact, policy implications, economic significance
✅ **Executive Briefing Format**: Structured daily briefing for decision-makers
✅ **Indonesian News Sources**: Kompas.com, Republika.co.id, Detik.com, Antara News, Tempo.co

## Examples Available

The skill includes comprehensive examples showing:

- **Complete Daily Briefing**: Real-world example with 5 top stories and analysis
- **Executive Summary**: Concise business outlook format
- **Business Impact Analysis**: Detailed sector-specific implications
- **Policy Analysis**: Regulatory framework assessment

These examples demonstrate best practices for progressive disclosure and professional news analysis, helping users understand how to structure and interpret different types of news briefings for various business contexts.
