# Global Digital Health Intelligence Skill

Provides comprehensive intelligence on digital health developments globally, with special focus on Asia-Pacific region and Indonesia-specific developments. Perfect for healthcare executives, policy makers, digital health innovators, and investors seeking market intelligence across global and emerging markets.

## Files Structure

```
global-digital-health-intelligence/
├── SKILL.md                    # Main skill definition and workflow
├── README.md                      # This documentation
├── references/                     # Reference materials and examples
│   ├── digital-health-sources.md      # Authoritative sources guide
│   ├── market-intelligence-framework.md    # Analysis methodologies
│   ├── indonesia-health-ecosystem.md  # Indonesia-specific context
│   └── technology-trends.md           # Global digital health trends
└── examples/                         # Progressive disclosure examples
    ├── sample-briefing-2025-01.md          # Recent intelligence example
    ├── indonesia-analysis-example.md           # Indonesia-specific analysis example
    └── regional-snapshot-example.md           # Asia-Pacific overview example
```

## Usage

1. **Upload Skill**: Install `global-digital-health-intelligence.zip` via Claude Desktop Settings → Capabilities → Skills → Upload skill

2. **Request Intelligence**:
   - "Update saya tentang perkembangan digital health global dan di Indonesia"
   - "Analisis persaingan telemedicine Indonesia terkini"
   - "Strategic assessment market digital health Asia Pasifik"

3. **Load Examples**: When helpful, request specific examples:
   - "Load digital health sources guide"
   - "Show sample briefing January 2025"
   - "Load technology trends reference"

## Skill Capabilities

✅ **Multi-Scope Intelligence**: Global trends + Asia-Pacific focus + Indonesia-specific analysis
✅ **Strategic Analysis**: Market impact, technology implications, regulatory assessment
✅ **Time-Frame Adaptation**: Intelligence gathering for recent developments (7 days), trends (30 days), strategic overview (90 days)
✅ **Executive Reporting**: Professional briefing format with strategic implications
✅ **Authoritative Sources**: WHO, HIMSS, regional health authorities, government agencies
✅ **Regional Context**: Asia-Pacific ecosystem dynamics affecting Indonesian market

Perfect for strategic digital health market intelligence and competitive analysis across global and emerging markets.
