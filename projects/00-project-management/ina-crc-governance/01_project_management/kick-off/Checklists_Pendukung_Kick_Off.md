## 6. Contingency Planning Checklist

### 6.1 Technical Issues

**[ ] Equipment Failure**
- [ ] Backup projector available
- [ ] Alternative laptop ready
- [ ] Mobile hotspot untuk internet backup
- [ ] Printed handouts sebagai backup digital

**[ ] Online Platform Issues**
- [ ] Backup platform ready (Google Meet)
- [ ] Technical support on speed dial
- [ ] Alternative communication methods
- [ ] Delay tolerance built into agenda

### 6.2 Participant Issues

**[ ] Key Stakeholder Absence**
- [ ] Backup delegates identified
- [ ] Pre-aligned decisions available
- [ ] Reschedule critical decisions
- [ ] Document absence dan follow-up plan

**[ ] Low Attendance**
- [ ] Minimum required participants identified
- [ ] Go/No-go decision criteria
- [ ] Rescheduling options prepared
- [ ] Communication plan untuk postponement

### 6.3 Content Issues

**[ ] Discussion Goes Off-Track**
- [ ] Strong facilitator assigned
- [ ] Parking lot untuk off-topic discussions
- [ ] Timekeeper with authority
- [ ] Clear decision framework

**[ ] No Decisions Made**
- [ ] Pre-meeting stakeholder alignment
- [ ] Clear decision matrix prepared
- [ ] Escalation path defined
- [ ] Follow-up meeting scheduled
