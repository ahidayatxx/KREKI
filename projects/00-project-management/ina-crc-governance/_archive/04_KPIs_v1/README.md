# 📊 Key Performance Indicators (KPI) INA-CRC

## 🚨 Sistem Alerting & Escalation

### **Threshold Levels**
- 🟢 **Normal (80-100%):** On track, no action required
- 🟡 **Warning (60-79%):** Monitor closely, corrective action planned
- 🔴 **Critical (<60%):** Immediate action required, management escalation

### **Alert Channels**
- **Dashboard Alerts:** Real-time visual notifications
- **Email Notifications:** Automatic alerts untuk responsible PICs
- **Mobile Push:** Critical alerts untuk on-the-go access
- **SMS Alerts:** Emergency notifications untuk critical issues

4. **Implementation:** Eksekusi improvement initiatives

1. **Strategic Review:** Evaluasi kesesuaian dengan tujuan strategis
