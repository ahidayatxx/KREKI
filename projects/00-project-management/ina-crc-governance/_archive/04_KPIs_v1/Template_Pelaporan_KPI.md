## 📋 Template Overview

Template ini menyediakan format standar untuk pelaporan KPI INA-CRC secara berkala. Template dirancang untuk memastikan konsistensi, akurasi, dan kemudahan analisis trend kinerja dari waktu ke waktu.

**Jenis Pelaporan:**
- 📊 **Dashboard Real-time:** Monitoring harian dan real-time
- 📈 **Bulanan:** Detail monthly performance analysis
- 📊 **Triwulanan:** Strategic quarterly review
- 📅 **Tahunan:** Comprehensive annual assessment

## 🔧 Cara Penggunaan Template

### 1. Persiapan Data
- Kumpulkan data dari semua sumber yang relevan
- Validasi akurasi dan kelengkapan data
- Hitung achievement percentage untuk setiap KPI
- Tentukan status menggunakan traffic light system

### 2. Penentuan Status
- 🟢 **Hijau (On Target/Exceeding):** ≥80% achievement
- 🟡 **Kuning (At Risk):** 60-79% achievement
- 🔴 **Merah (Critical):** <60% achievement

### 3. Analisis dan Insight
- Identifikasi pola dan trend dari data historis
- Analisis penyebab deviasi dari target
- Generate actionable insights dan recommendations
- Siapkan naratif executive summary

### 4. Review dan Approval
- Review draft report dengan team
- Validasi accuracy calculations dan interpretations
- Obtain approval dari manajemen sebelum distribusi
- Archive report untuk future reference

---

## 📋 Best Practices Penggunaan

### 🎯 Data Visualization
- Gunakan chart dan grafik untuk memudahkan pemahaman
- Konsisten warna dan format untuk setiap laporan
- Highlight area yang memerlukan perhatian khusus
- Include comparison data untuk konteks

### 📊 Storytelling dengan Data
- Mulai dengan executive summary yang jelas
- Gunakan narrative untuk menjelaskan angka dan trend
- Include success stories dan lessons learned
- End dengan clear next steps dan action items

### ⚠️ Quality Assurance
- Double-check semua calculations dan percentages
- Validasi source data dan assumptions
- Review untuk consistency dengan previous reports
- Ensure action items SMART (Specific, Measurable, Achievable, Relevant, Time-bound)
